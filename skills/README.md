# Skills 目录 - Claw 智能体的武器库

> 这个目录用于存放可复用的工具、脚本和配置文件，提高工作效率。

## 使用规则
1. **分类存放** - 按功能模块或用途分类管理文件
2. **文档完整** - 每个工具都应有清晰的说明和使用指南
3. **版本管理** - 重要工具应记录版本号和更新日志
4. **优先复用** - 接到新任务时，先检查本目录是否有可用工具

## 目录结构规划
```
skills/
├── automation/      # 自动化脚本和部署配置
├── code-generation/ # 代码生成相关工具
├── optimization/    # 性能优化和 SEO 工具
├── integration/     # 外部集成和 API 调用
└── utilities/       # 通用工具函数
```
